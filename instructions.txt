Step 1: First include passport
    > composer require laravel/passport

Step 2: Include the service provider in app.php
    Laravel\Passport\PassportServiceProvider::class,

Step 3: Run migrations
    php artisan migrate:refresh --seed

Step 4: Run passport install command
    php artisan passport:install

Step 5: 
    In model we added HasApiTokens class of Passport:
        use Laravel\Passport\HasApiTokens;
        ...
        use HasApiTokens, Notifiable;
        ...
    
    In AuthServiceProvider we added “Passport::routes()”:
        use Laravel\Passport\Passport;
        ...
        boot()
            Passport::routes();
        ...

    In auth.php, we added api auth configuration.
        'api' => [ 
            'driver' => 'passport', 
            'provider' => 'users', 
        ],
